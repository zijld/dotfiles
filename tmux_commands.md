# tmux cheatsheet

start new:

    tmux

list sessions:

    tmux ls

kill session:

    tmux kill-session -t myname

In tmux, hit the prefix `ctrl+b` and then:

## Sessions

    :new<CR>  new session
    s  list sessions
    $  name session

## Windows (tabs)

    c           new window
    ,           name window
    w           list windows
    f           find window
    &           kill window

## Panes (splits)

    %  horizontal split
    "  vertical split

    o  swap panes
    q  show pane numbers
    x  kill pane
    ‚çΩ  space - toggle between layouts
